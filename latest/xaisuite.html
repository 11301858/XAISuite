xaichooser Documentation

xaisuite.xaichooser.train_and_explainModel(model, tabular_data, x_ai, indexList=[], scale=True, scaleType='StandardScaler', addendum='', verbose=False, **modelSpecificArgs)

A function that attempts to train and explain a particular sklearn model.

Parameters

  • model (str) – Name of Model

  • tabular_data (Tabular) – Tabular object representing data set to be used in training

  • x_ai (list) – List of explanatory models to be used

  • indexList (list) – Specific test data instance to be explained, by default empty (indicating all instances should be explained)

  • scale (bool) – Whether data should be scaled before training, by default True

  • scaleType (str) – Default Scaler type is “StandardScaler”. Example: Use “MinMaxScaler” for MultinomialNB model.

  • addendum (str) – Added string to explanation files in case multiple models are being trained and explained within the same directory, to prevent overwriting. By default, empty string.

  • verbose (bool) – Whether debugging information should be printed, by default False

  • **modelSpecificArgs – Specific arguments to pass on to the model function

Returns

The learning model and The scaler (if applicable) if user wants to predict more values. In List format

Return type

list
